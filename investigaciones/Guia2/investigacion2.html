Método push() y pop()

  • Guías de JS
  • Investigaciones

En javascript podemos hacer uso de las mátrices o arreglos, los cuales nos ayudan a obtener no solo muchos datos, sino que también de cualquier tipo además de estos no es necesario que su tamaño no seán predefinido si es que no se quiere, esto debido a la existencia de diferente tipos de metodos los cuales ayudan a ingresar más datos sin ningún problema


EL MÉTODO PUSH()

Este método ayuda a la creación de arreglos en los cuales no se tiene por seguro el tamaño del arreglo, para ello el método push() que es llamado Array.push(dato1, dato2, ...) es de tener en cuenta que para usarlo es recomendado el usar más de 1 dato a ingresar.

var sports = ['soccer', 'baseball'];
var total = sports.push('football', 'swimming');

console.log(sports); // ['soccer', 'baseball', 'football', 'swimming']
console.log(total); // 4


Método pop()

En este caso para modificar la longitud o tamaño de nuestro arreglo podemos hacer uso del método pop() el cual, elimina el último dato que nuestro array tiene devolviendolo, pero si se vuelve a escribir el array podremos ver que ese dato ha sido eliminado del todo

var myFish = ['angel', 'clown', 'mandarin', 'sturgeon'];
var popped = myFish.pop();
console.log(myFish); // ['angel', 'clown', 'mandarin' ]
console.log(popped);//'sturgeon'


EJEMPLO CON AMBOS MÉTODOS

var colors = ['blue', 'red', 'green', 'yellow']
colors.push('White', 'Black');//agregamos dos nuevos datos
console.log(colors)//'['blue', 'red', 'green', 'yellow', 'White', 'Black']'
colors.pop()//Borramos el campo de 'Black'
console.log(colors)//'['blue', 'red', 'green', 'yellow', 'White']'
