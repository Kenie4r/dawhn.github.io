Una de las principales funciones necesarias para poder trabajar ya sea con números enteros o números flotantes, para ello en es necesario un objeto llamado Math(), el cual básicamente un clase la cual está instalada de manera predeterminada en JavaScript

Cómo hemos visto la intención de esto es tener un clase en la cual podremos tener todos las funciones y también atributos, entre estos, el Número PI, Número de Euler , etc. En este caso vamos a tener en cuenta las funciones donde pondremos un ejemplo y para que sirve este:


Función Descripción Código
Math.abs() Este nos retornar el valor abosuluto sus parametros son Math.abs(x) donde x es el número.

//usaremos la fúnción del valor absoluto
var myNum = -34;
var absoluto = Math.abs(myNum);
console.log(absoluto); //Nos devolvera 34 ya que este es su valor absoluto

Math.round() Nos devuelve el número redondeado, siendo el resultado el número más cercano. La sintaxis es: Math.round(x) siendo x el número.

//usaremos la fúnción para redondear
var myNum = 34.23;
var redondeado = Math.round(myNum);
console.log(redondeado); //Nos devolvera 34

Math.ceil() Esta nos devuelve el mayor número que sigue después del número entregado, los parametros son Math.ceil(x) donde x es un número.

Math.ceil(.95); // 1
Math.ceil(4); // 4
Math.ceil(7.004); // 8
Math.ceil(-0.95); // -0
Math.ceil(-4); // -4
Math.ceil(-7.004); // -7

Math.floor() Nos devuelve el menor número de al entregado los parametros son Math.floor(x) donde x es un número

Math.floor(0.223);//nos devuelve 0

Math.exp() Este nos devuelve el resultado del exponncial de ex Los parametros son Matn.exp(x) donde x es un número

Math.exp(-1); // 0.36787944117144233
Math.exp(0); // 1
Math.exp(1); // 2.718281828459045

Math.log() Nos devuelve el valor de el logaritmo natural de x número, los parámetros son Math.log(x) donde x es un número

Math.log(1); // 0 Math.log(10); // 2.302585092994046

Math.random() Nos devuelve un número random entre 0 y 1 (sin tomar al 1), este no obtiene ningún parametro

// Retorna un número aleatorio entre min (incluido) y max (excluido)
function getRandomArbitrary(min, max) {
 return Math.random() * (max - min) + min;
}
